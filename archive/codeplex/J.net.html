J.net

J.net is a simple API for using .net with Java like API.

Differents

  • Object
    • Method clone is public.
  • java.io.FilterOutputStream
    • out as C# keywort is replaced by outJ
  • java.lang.System
    • Class is renamed to SystemJ, because package is called System.
    • out as C# keywort is replaced by outJ
  • java.lang.ref
    • Package is renamed to refj
  • java.nio.ByteBuffer
    • Variables mark, positions and limit renamed to *J, because same named methods have public visiblity.
  • java.util.Dictionary
    • This Java abstract class with only abstract methods is changed to interface type.
  • java.util.Map
    • The package Map is renamed to MapNS (for namespace)
